The Field Database for Corrosion Inhibitor Application (FD-CIA) system is a digital solution for corrosion inhibitor selection. It provides a customized template to upload the parameters of systems with successful corrosion inhibitor applications. This information is then easily uploaded to the Azure database and analyzed by our statistical functions to visualize the data. The collected historical data and machine learning module deployed on the LEGO web app instantly provides artificial intelligence recommendations to guide product selection or support a proposed application.
